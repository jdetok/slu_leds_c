# slu_leds master repo

## repo organization
- ## BOM.md
    - bill of materials list with exact parts used
- ## INSTRUCTIONS.md
    - user instructions
- ### pcb/ 
    - contains KiCAD project with shematic/PCB files 
- ### firmware/
    - contains C++ firmware that gets uploaded to the arduino
    - project built with [PlatformIO](https://platformio.org/) [VSCode extension](https://platformio.org/install/ide?install=vscode)
